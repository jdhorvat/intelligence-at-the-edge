# Contributing

This is a living document released to the public domain.

## How to engage

- **Found a flaw in the reasoning?** Open an issue. Be specific about which section and what the flaw is.
- **Want to extend the architecture?** Fork it. Build on it. No permission needed.
- **Disagree with a design decision?** Open an issue with your alternative and the reasoning behind it.
- **Built something based on this?** A mention is appreciated but not required.

## What this is not

This is not a software project. There is no code to contribute. It is a design document — a set of architectural ideas offered freely.

## Tone

Arguments over ideas are welcome. Keep it about the architecture.
